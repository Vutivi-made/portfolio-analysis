# Portfolio Simulation Tool

## Description

Simple Python project to simulate portfolio allocations and analyzed historical data to identify trends and investment opportunities.

You can enter multiple stock tickers, choose a date range, and assign weights to each stock.

## Features

- Fetch historical stock data from Yahoo Finance
- Compute daily returns and cumulative portfolio growth
- Calculate moving averages for trend analysis
- Simulate portfolio allocations with custom weights
- Compute basic portfolio metrics: expected return and volatility
- Visualize portfolio growth with a simple line chart

## Requirements

- Python3
- pandas
- numpy
- matplotlib
- yfinance
